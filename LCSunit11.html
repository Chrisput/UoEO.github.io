Unit 11: Advanced SQL

On completion of this unit I will be able to

  • Write complex queries to extract the results from a database using SQL.
  • Apply aggregate, arithmetic, date and string functions in SQL for summarised reporting.
  • Evaluate query results.

Reflections

Below are my key learnings and reflections

  • This Unit my entire focus has been on learning and practising SQL
  • DDL (Data Definition Language): CREATE, DROP, ALTER, TRUNCATE
  • DQL (Data Query Language): SELECT
  • DML (Data Manipulation Language): INSERT, UPDATE, DELETE
  • Aggregate functions: COUNT, SUM, AVG, MIN, MAX
  • Constraints: NOT NULL, UNIQUE, PRIMARY KEY, FOREIGN KEY, CHECK
  • Joins: INNER JOIN, LEFT JOIN, RIGHT JOIN, FULL JOIN
