Induction

Prior to commencing with Module 1 - Launching into Computer Science, an induction was held to welcome students to the course, to provide an opportunity to familiarise with the Virtual Learning Environment, to get to know other students, and to settle into the studies.

Learning outcomes

I will have learned:

  • to demonstrate the technical and personal requirements of online learning.
  • to demonstrate competencies in the use of the tools available to online learners.
  • to demonstrate academic skills, e.g. essay/report writing, research techniques, referencing, reflective learning and critial thinking.

Reflections

I found the induction helpful and in particular I learned

  • how to navigate GitHub at a basic level, both through using the Command Line and through the GitHub user interface
  • some basic syntax for HTML and Markdown
  • to understand the guidelines for academic writing
  • how to use the Harvard Referencing for academic writing
  • how to contribute to a discussion forum
