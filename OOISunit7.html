Unit 7: Database Design

On completion of this unit I am able to

  • Develop the knowledge and skills to apply database design principles
  • Design a database that has been correctly normalised

Reflections

Below are my key learnings and reflections

  • Understand what a relational database is: compare SQL vs. NoSQL databases; relational database concepts; DBMS features and engines comparison; and SQL data types
  • Design a logical model: identify database integrity constraints; and normalise to the First (1NF), Second (2NF), and Third (3NF) normal Forms
  • Practicej MySQL operators: UNION and INTERSECT
