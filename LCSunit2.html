Unit 2: Introduction to Computer Architecture and the Functionality

On completion of this unit I will be able to

  • Appreciate the evolution of computer and computer technologies due to innovations
  • Decribe the critical functional issues of computer structure components
  • Identify and demonstrate the basic characteristics, functions and features of each element of a computer system
    • Demonstrate the theories and nature of data representation for computers

Reflections

Below are my key learnings and reflections

  • Work with Boolean operations and construct flipflop circuits: AND (times), OR (plus), XOR, NOT
  • Understand types and advantages of mass storage devices: magnetic systems (slow speed, long search times, low cost & high data capacity -> good for data archival), optical systems (long continuous strings, multiple layers, not good for random access to data -> ideal for music and video, and flash drives(eroding memory chambers, resilient to shock, portable, high capabity -> good for data portability).
  • Convert from binary to decimal notation and vica versa. Integers (two's complement notation with sign bit) and floating-point.
  • CPU components and function: arithmic/logic unit, control unit, registers, bus, main memory. The stored-program concept.
  • Understand how machine instructions are decoded and executed, and how the CPU communicates with other devices.
  • Learn HTML syntax to build my e-Portfolio, e.g. how to link pages, how to create menu's, how to include pictures and links, etc.
