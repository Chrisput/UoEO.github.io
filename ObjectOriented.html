Object-oriented Information Systems - In Progress

This module teaches how to contextualise and develop information systems using object-oriented approaches with various programming languages, such as Phyton and SQL.

Learning outcomes

This module has enabled me:

  • to appraise and evaluate critically the concepts and principles of information systems.
  • to design or modify and document an object-oriented information system using appropriate tools.
  • to develop an object-oriented information system design, implementing this knowledge in applicable programming languages, such as Python and SQL.
  • to develop, implement and evaluate critically information system solutions to facilitate business decisions.

Tutor: Beran Necat

Core reading books for this module:

  • Phyton 3 Object-Oriented Programming by D. Phillips
  • Database Systems: A Practical Approach to Design, Implementation and Management by T.M. Connolly & C.E. Begg

Unit 1: Introduction to Information Systems

This unit provides an introduction to the core concepts of the module and offers context around information systems and their development. It begins by focusing on the key terms and concepts, followed by looking at some notable failures in information systems. The unit introduces the Systems Development Lifecycle, its phases and how it fits with systems development.

  • See details of learning in Unit 1

Unit 2: Information Systems and their Importance

This unit focusses on the individual elements of an information systame, it investigates how the elements of an information system fit together; and discusses potential issues that can be encountered by an information system.

  • See details of learning in Unit 2

Unit 3: Fundamentals of Object-Oriented Design

This unit provides an introduction to object-oriented design. It begins by introducing the core concepts around classes and objects, followed by the key principles of object-oriented design.

  • See details of learning in Unit 3

Unit 4: Object-oriented Developement and Python

This module provides a practical introduction to object-oriented programming in Python.

  • See details of learning in Unit 4

Unit 5: Understanding UML

This unit provides an overview of Unified Modelling Language, which introduces the key elements to document and design a system.

  • See details of learning in Unit 5

Unit 6: Hands-on with UML

This unit provides a hands-on approach to UML, putting into practice knowledge gained in the previous unit. Additionally, the focus is on how to identify classes for an information system.

  • See details of learning in Unit 6

Unit 7: Database Design

This unit introduces database technologies and provides an understanding of the fundamental concepts and principles to successfully design a database.

  • See details of learning in Unit 7

Unit 8: Hands-on with Database Design

This unit provides a hands-on approach to database design, putting into practice the knowledge and skills gained in the previous unit. It also offers an opportunity to develop entity relationship diagrams (ERD).

Unit 9: Implementing Database with SQL
